Genome Resources

Genome Assembly and Annotation Data

Genome files are hosted in external datasets. Each table entry links directly to the corresponding downloadable file.

This page provides genome assembly, annotation, CDS, and proteome resources generated from genome sequencing and downstream annotation workflows. Assembly files are provided in FASTA format, annotation files in GFF format, and sequence-derived CDS and protein files in FASTA format.

Large files are stored outside this website to keep the portfolio lightweight. After the external dataset is ready, replace the placeholder URLs in assets/js/data.js with the final links.
